# MS1005 TDC驱动程序工程

## 项目概述

本项目是一个基于STM32F103C8微控制器的MS1005时间数字转换器(TDC)驱动程序。MS1005是一款高精度的时间测量芯片，主要用于飞行时间(TOF)测量、脉冲宽度测量等应用场景。

## 硬件平台

- **主控芯片**: STM32F103C8 (Cortex-M3内核)
- **目标器件**: MS1005 TDC芯片
- **通信接口**: SPI (STM32 SPI1)
- **调试接口**: J-Link
- **串口通信**: USART1 (115200波特率)

## 工程结构
- `User/`: 包含MS1005驱动程序和主程序代码
  - `ms1005_driver.c`: MS1005驱动实现文件
  - `ms1005_driver.h`: MS1005驱动头文件
  - `main.c`: 主程序入口文件

## 主要功能特性

### 1. MS1005驱动功能
- **硬件初始化**: GPIO配置、SPI配置、中断配置
- **通信协议**: SPI全双工通信，支持读写操作
- **工作模式**: 支持校准模式和非校准模式
- **测量功能**: 飞行时间(TOF)测量、脉冲宽度测量
- **错误处理**: 状态寄存器读取和错误检测

### 2. 系统功能
- **时钟配置**: 72MHz系统时钟，HSE外部晶振
- **定时器**: Timer2用于系统延时(1ms精度)
- **串口通信**: USART1用于调试输出和命令输入
- **中断管理**: 外部中断处理MS1005中断信号

## 硬件连接

### MS1005引脚连接
| MS1005引脚 | STM32引脚 | 功能描述 |
|-----------|-----------|----------|
| SSN       | PA4       | SPI片选 |
| SCK       | PA5       | SPI时钟 |
| MISO      | PA6       | SPI数据输入 |
| MOSI      | PA7       | SPI数据输出 |
| INTN      | PB1       | 中断信号输入 |
| RSTN      | PB0       | 复位信号 |
| CAL       | PB15      | 校准信号 |
| NCAL      | PB13      | 非校准信号 |

### 测试模式引脚(可选)
| MS1005引脚 | STM32引脚 | 功能描述 |
|-----------|-----------|----------|
| START     | PA11      | 启动脉冲 |
| STOP1     | PA12      | 停止脉冲1 |
| STOP2     | PA12      | 停止脉冲2 |

## 使用说明

### 1. 编译环境
- **IDE**: Keil MDK-ARM uVision
- **编译器**: ARMCC V5.06 update 7
- **设备包**: STM32F1xx_DFP 2.4.1

### 2. 编译步骤
1. 打开 `MS1005_Driver.uvprojx` 工程文件
2. 选择目标设备: STM32F103C8
3. 设置编译选项(已预配置):
   - 优化级别: Level 0 (调试模式)
   - 使用微库(MicroLIB)
   - C99标准
4. 编译工程生成HEX文件

### 3. 程序运行
1. 通过J-Link将程序下载到STM32
2. 连接串口终端(115200波特率)
3. 系统启动后显示提示信息:

### 4. 操作命令
- **模式选择**: 输入1-4选择测量模式
- **采样次数**: 输入0-65535设置采样数量
- **开始测量**: 按Enter键开始测量

## 技术特点

### 1. 性能优化
- **DWT计数器**: 使用Cortex-M3 DWT周期计数器实现微秒级超时检测
- **环形缓冲区**: 串口接收使用环形缓冲区，提高数据接收效率
- **中断优化**: 最小化中断服务程序执行时间

### 2. 代码质量

## 注意事项

1. **时钟配置**: 确保外部晶振正常工作，系统时钟配置为72MHz
2. **SPI时序**: MS1005对SPI时序有严格要求，已通过延时保证
3. **中断处理**: INTN中断为下降沿触发，确保硬件连接正确
4. **电源稳定性**: MS1005对电源噪声敏感，建议使用稳定的电源
